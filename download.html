CeMS Free Ready?

CeMS 무료버전은 조건 없이 완전히 평생 무료입니다.

  • 제약

    CeMS 를 설치한 곳과 동일한 위치에서만 접속이 가능합니다. (http://127.0.0.1:2014)

    최적화된 기본 설정으로 동작하며 설정 변경이 불가합니다.

  • 준비

    윈도우즈 실행파일로 제공됩니다.

    시스템 속성 >> 환경변수 >> 시스템 변수 항목에 자바 설치 경로를 입력해 주세요. 변수명 JAVA_HOME 값 C:\Program Files\Java\jdk-12.0.1 (예시)

    Java는 OpenJDK 12 버전이 가장 안정적입니다.

    CeMS 를 실행하기 위해 폴더를 생성하고 좌측 상단의 Download 버튼을 우클릭 하여 생성된 폴더에 실행파일(CeMSFree.exe)을 저장합니다.

    실행파일을 실행시키면 폴더 내 data 폴더가 생성되고 CeMS 가 수집하는 모든 데이터가 이 곳에 저장됩니다.

    CeMS 가 정상적으로 시작되면 이 페이지는 더 이상 보이지 않습니다.

  • 주의

    CeMS 를 종료 하고자 할 때는 반드시! 콘솔창에서 Ctrl + C 클릭으로 정상 종료하세요. 비정상 종료 하는 경우 데이터베이스를 복구하지 못할 수도 있습니다.

    CeMS 가 모니터링하고자 하는 대상 노드 사이에 방화벽이 존재한다면 UDP 161 포트를 허용해 주어야 합니다.
